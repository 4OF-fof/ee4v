# ee4v (EditorExtensions for VRChat)

## 概要
VRChatクリエイター向けのUnityエディタ拡張ツールセットです。独自のアセット管理システム、BOOTH購入履歴との連携機能、HierarchyやProjectウィンドウのUI拡張機能を提供し、制作環境の効率化を目的としています。

## 主な機能

### Asset Manager (アセットマネージャー)
フォルダやタグを使用して所有アセットを整理・管理する機能です。
- インポート機能: 管理下にあるアセット（UnityPackageやZIPファイル）を、ワンクリックでプロジェクトへインポート可能。依存アセットも自動インポート
- フォルダ管理: アセットを仮想的なフォルダでグループ化
- タグ機能: アセットにタグを付与して分類可能
- アセット情報: 各アセットに独自の説明文や依存関係を記録
- 高度なソート: 名前、追加日、更新日、サイズ、ファイルタイプでの並び替え
- フィルタリング: タグやフォルダによる絞り込み

### BOOTH ライブラリ連携
Webブラウザ上のBOOTH購入履歴（ライブラリ）情報をUnityエディタに取り込む機能です。
- 同梱のユーザースクリプト（BoothLibraryExport.user.js）を使用
- BOOTHのライブラリページから購入済みアイテムの情報を取得
- Unity上で購入済みアイテムをリスト化し、boothからのダウンロードやインポートが可能

### エディタUI拡張
Unity標準のウィンドウ機能を強化し、視認性と操作性を向上させます。

**Hierarchy（ヒエラルキー）拡張**
- コンポーネント表示: オブジェクトにアタッチされている主要なコンポーネントのアイコンを表示。アイコンをクリックすると、そのコンポーネントのインスペクタを別ウィンドウで展開して編集が可能
- 階層ライン表示: 親子関係を視認しやすくするデプスライン（罫線）の描画
- シーンスイッチャー: プロジェクト内のシーンへ素早く切り替えられるメニュー機能
- オブジェクトの非表示と管理: スタイル設定（アイコン・色変更）を行うウィンドウから、オブジェクトを「Hierarchyから隠す」ことが可能。非表示化されたオブジェクトは専用リストで管理され、そこから復元できる

**Project（プロジェクト）拡張**
- フォルダスタイル変更: フォルダごとにアイコンや色を設定し、重要なフォルダを強調表示
- ワークスペース（タブ）機能: よく使うフォルダやアセットへのショートカットをタブとして登録し、素早くアクセス可能

### その他の機能
- 自動バックアップ (AutoBackup)
- 多言語対応 (日本語、英語、韓国語)

## 使い方

### 1. Unityへの導入
本パッケージをUnityプロジェクトに導入します。`package.json`が含まれているため、VCCやALCOMなどを利用してプロジェクトに追加することもできます。Unity 2022.3での動作が想定されています。

### 2. BOOTH連携のセットアップ
BOOTHの購入履歴をUnityに取り込むには、Webブラウザ側の準備が必要です。

1. Webブラウザにユーザースクリプトマネージャー（Tampermonkey、Violentmonkeyなど）をインストールします。
2. リポジトリ内の `~UserScript/BoothLibraryExport.user.js` をブラウザにインストールします。

### 注意事項
- BOOTH連携時は、ブラウザのタブとUnityを閉じずに処理の完了を待ってください。
