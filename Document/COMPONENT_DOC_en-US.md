# Unity Editor Extension Component List

## Hierarchy Extensions

| Name | Description | Trigger |
| :--- | :--- | :--- |
| **Component Icons** | Displays icons for components attached to the object. | Always (Click to open window) |
| **GaneObject Window** | Expands a window for customizing object appearance and displaying information. | Click the menu icon |
| **Hidden Object** | Displays a list of hidden objects. | Click the eye icon |
| **DepthLine** | Displays guide lines in the hierarchy according to object depth. | Always |
| **CustomStyle: Heading** | Applies the appearance of a heading to the object. | Add a specific string to the beginning of the name (Default: ':HEADING') |
| **Scene Switcher** | Expands a window for easy switching between scenes. | Click the scene name |
| **CustomStyle: Separator** | Applies the appearance of a separator line to the object. | Add a specific string to the beginning of the name (Default: ':SEPARATOR') |
| **Styled Object** | Sets colors and icons for objects to improve visibility in the hierarchy. | Set from dedicated window (GaneObject Window) |

## Project Extensions

| Name | Description | Trigger |
| :--- | :--- | :--- |
| **Folder Overlay** | Displays overlay icons for items contained in the folder. | Always |
| **Styled Folder** | Sets colors and icons for folders to improve visibility in the Project window. | Alt key on folder |

## Project Toolbar Extensions

| Name | Description | Trigger |
| :--- | :--- | :--- |
| **Tab** | Adds a tab to the project window. | Click the Add button, or drag and drop a folder into the tab area. |
| **WorkSpace** | Creates a temporary workspace and uses it as a virtual folder. | Right-click the Add button. You can add assets by dragging and dropping them onto the created tab. |

## Editor Service Extensions

| Name | Description | Trigger |
| :--- | :--- | :--- |
| **Auto Backup** | Automatically create backups of your avatars. | When uploading to VRChat is successful. |

## Editor Utility Extensions

| Name | Description | Trigger |
| :--- | :--- | :--- |
| **Variant Converter** | Creates a variant of the avatar prefab. Materials are also converted into variants at the same time. | Right-click the prefab in the Hierarchy. |
| **Material List** | Displays a list of materials used in the prefab. | Open the window from ee4v > Material List. Select a prefab in the Hierarchy or specify a target in the window. |
