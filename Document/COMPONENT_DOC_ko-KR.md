# Unity Editor 확장 기능 컴포넌트 목록

## Hierarchy 확장 기능

| 이름 (Name) | 기능 (Description) | 작동 조건 (Trigger) |
| :--- | :--- | :--- |
| **Component Icons** | 오브젝트에 첨부된 컴포넌트의 아이콘을 표시합니다. | 항상 (클릭 시 창 확장) |
| **GaneObject Window** | 오브젝트 모양 사용자 정의 및 정보 표시 창을 확장합니다. | 메뉴 아이콘 클릭 |
| **Hidden Object** | 숨겨진 오브젝트 목록을 표시합니다. | 눈 아이콘 클릭 |
| **DepthLine** | 오브젝트 깊이에 따라 계층 구조 내에 안내선을 표시합니다. | 항상 |
| **CustomStyle: Heading** | 제목의 모양을 오브젝트에 적용합니다. | 이름의 시작 부분에 특정 문자열 추가 (기본값: ':HEADING') |
| **Scene Switcher** | 씬 간 전환을 쉽게 할 수 있는 창을 확장합니다. | 씬 이름 클릭 |
| **CustomStyle: Separator** | 구분선의 모양을 오브젝트에 적용합니다. | 이름의 시작 부분에 특정 문자열 추가 (기본값: ':SEPARATOR') |
| **Styled Object** | 오브젝트에 색상과 아이콘을 설정하여 계층 구조에서 가시성을 향상시킵니다. | 전용 창 (GaneObject Window)에서 설정 |

## Project 확장 기능

| 이름 (Name) | 기능 (Description) | 작동 조건 (Trigger) |
| :--- | :--- | :--- |
| **Folder Overlay** | 폴더에 포함된 항목의 아이콘을 오버레이 표시합니다. | 항상 |
| **Styled Folder** | 폴더에 색상과 아이콘을 설정하여 프로젝트 창에서 가시성을 향상시킵니다. | 폴더에서 alt 키 |

## Project Toolbar 확장 기능

| 이름 (Name) | 기능 (Description) | 작동 조건 (Trigger) |
| :--- | :--- | :--- |
| **Tab** | 프로젝트 창에 탭을 추가합니다. | 추가 버튼을 클릭하거나 탭 영역에 폴더를 드래그 앤 드롭합니다. |
| **WorkSpace** | 임시 작업 공간을 생성하여 가상 폴더처럼 사용합니다. | 추가 버튼을 마우스 오른쪽 버튼으로 클릭합니다. 생성된 탭에 에셋을 드래그 앤 드롭하여 추가할 수 있습니다. |

| 이름 (Name) | 기능 (Description) | 작동 조건 (Trigger) |
| :--- | :--- | :--- |
| **Auto Backup** | 만든 아바타의 백업을 자동으로 만듭니다. | VRChat에 업로드 성공 시 |
